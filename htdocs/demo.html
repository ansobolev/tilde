Welcome to Tilde, the knowledge-management system for computational materials science. In a nutshell, it is an application, which creates the systemized repository from raw scientific modeling data.

Tilde is an open-source project of Max-Planck Institute for Solid State Research (Stuttgart, Germany) and Saint Petersburg State University (Russia). See more info at http://wwwtilda.googlecode.com
